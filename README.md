## Profile

Backend Software Engineer specialising in designing, building, and scaling high-performance **backend systems and applications**.

What i do:
- Secure APIs and data models
- Design for idempotency and consistency
- Implement retry, rollback, and reconciliation strategies
- Build queue-based background processing systems
- Ensure observability and traceability in all systems


---

## Systems I Build

- Ledger systems (double-entry, audit-safe)
- Payment orchestration pipelines
- Stablecoin custody & settlement flows
- Rule-based financial automation engines
- Conversion layers (crypto ⇄ fiat)
- Queue-based background processing (retries, idempotency)
